# Fork Policy — cxplus playground

This repository is an experimental, personal fork of the OpenAI Codex CLI. It is not intended to be merged upstream into OpenAI’s Codex.

## Scope and expectations

- Purpose: sandbox for experiments used in the author's projects.
- Support: no official support; issues and timelines are best‑effort.
- Stability: breaking changes may occur without notice.
- Licensing: original licenses apply (Apache‑2.0 for upstream code unless otherwise noted).
- Branding: assets here (e.g., `codex-rs/logo*`) brand this fork (cxplus) and should not be confused with upstream branding.

## Upstream

- No merge‑back intent.
- Diffs may diverge significantly from upstream Codex.

## Contact

For questions about this fork specifically, open an issue in this repository. Do not file upstream issues for fork‑specific behavior.
